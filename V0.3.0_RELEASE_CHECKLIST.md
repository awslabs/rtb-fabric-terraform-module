## Key Features

### Inbound External Link
- Accepts connections from external RTB Fabric gateways
- Simplified log settings (flattened structure)
- Responder error masking support
- Standard Terraform map format for tags
- Full validation

### Configuration Example
```hcl
inbound_external_link = {
  create     = true
  gateway_id = "rtb-gw-abc123"
  
  link_log_settings = {
    error_log  = 10
    filter_log = 5
  }
  
  link_attributes = {
    customer_provided_id = "external-partner"
    responder_error_masking = [
      {
        http_code                   = "400"
        action                      = "NO_BID"
        logging_types               = ["METRIC", "RESPONSE"]
        response_logging_percentage = 15.0
      }
    ]
  }
  
  tags = {
    Environment = "Production"
    LinkType    = "External"
  }
}
```
